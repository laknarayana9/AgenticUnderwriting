# Property Eligibility Guidelines

## Basic Eligibility Requirements

### Property Types
**Acceptable:**
- Single-family detached homes
- Condominiums (max 4 stories)
- Townhouses (end units only)
- Manufactured homes (post-1976, permanent foundation)

**Not Acceptable:**
- Multi-family buildings (>4 units)
- Commercial properties
- Agricultural operations
- Properties used for business purposes

### Age and Condition
**Acceptable:**
- Properties built after 1940
- Well-maintained properties with recent updates
- Properties with updated electrical, plumbing, and roofing

**Requires Referral:**
- Properties built between 1900-1940
- Properties with visible deferred maintenance
- Properties with original systems (electrical, plumbing)

**Not Acceptable:**
- Properties built before 1900
- Properties with major structural issues
- Properties with documented code violations

### Coverage Limits

**Standard Limits:**
- Dwelling: Up to $500,000
- Other Structures: Up to 10% of dwelling limit
- Personal Property: Up to 70% of dwelling limit
- Loss of Use: Up to 20% of dwelling limit

**High-Value Properties:**
- Properties over $500,000 require supplemental application
- Appraisal may be required
- Additional underwriting documentation needed

### Required Documentation

**Always Required:**
- Property photos (exterior and interior)
- Basic property information (age, square footage, construction type)
- Declaration page from current insurance (if applicable)

**Conditionally Required:**
- Home inspection report (for properties >50 years old)
- Appraisal (for high-value properties)
- Repair estimates (if visible damage)

## Automatic Declines
- Properties with unrepaired damage
- Properties with active mold or water damage
- Properties with fire damage
- Properties in serious disrepair
